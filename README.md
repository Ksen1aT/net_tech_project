# Чат с отправкой файлов

Веб-чат, позволяющий обмениваться сообщениями и отправлять файлы (фото, документы) другим пользователям.

## Возможности

- Отправка текстовых сообщений
- Отправка файлов (изображения, PDF, текстовые файлы)
- Изображения показываются прямо в чате, по клику открываются в полном размере; документы отображаются в виде ссылки на скачивание
- Каждый пользователь выбирает себе имя, которое сохраняется в браузере и не сбрасывается при перезагрузке
- Свои сообщения выделяются розовым цветом, чужие сообщения — серые
- Новые сообщения подгружаются каждые 2 секунды без перезагрузки страницы


## Технологии

- Backend: Python + Flask
- Frontend: HTML, CSS, JavaScript

## Установка и запуск (локально)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Ksen1aT/net_tech_project.git
cd net_tech_project
git checkout dev
```


### 2. Создать виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install flask
```

### 4. Запустить сервер

```bash
python3 chat.py
```

### 5. Открыть в браузере

http://127.0.0.1:port
(где port - номер порта - указывается при запуске приложения в файле chat.py)
